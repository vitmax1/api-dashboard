Информация о странице

Эта страница использует трекер посещений.

При загрузке страницы автоматически:

  • Определяется тип устройства (Desktop/Mobile)
  • Получается город по IP-адресу
  • Данные отправляются на сервер

Как встроить трекер на свой сайт

Добавьте следующий код перед закрывающим тегом </body>:

<script
    src="http://localhost:8000/tracker.js"
    data-api-url="http://localhost:8000/api/track"
></script>

Параметры

  • src — URL файла трекера
  • data-api-url — URL API endpoint для приема данных
